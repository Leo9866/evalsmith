# Changelog

All notable changes to EvalSmith will be documented in this file.

## [0.1.0] - 2026-05-01

Initial open-source release.

### Added

- React web console for traces, datasets, experiments, annotations, monitoring, and settings
- Auth, trace, dataset, annotation, eval engine, and monitor services
- Python, Go, and TypeScript SDKs
- Docker Compose trial deployment
- Kubernetes deployment manifests
- Public API schema and usage/deployment documentation
- Open source guard, install check, and end-to-end validation scripts
- GitHub repository maintenance guidance and contribution templates

### Security

- Placeholder-only environment and secret examples
- Open-source export and guard checks
- Demo account documentation for local trial use only
